Cara Mendapatkan Gemini API Key Gratis

  1. Kunjungi situs Google AI Studio.
  2. Login dengan akun Google Anda.
  3. Klik tombol "Get API key" di pojok kiri atas.
  4. Pilih "Create API key in new project".
  5. Salin (copy) API key yang muncul di layar.
  6. Tempel (paste) API key tersebut ke dalam kolom di aplikasi ini.
